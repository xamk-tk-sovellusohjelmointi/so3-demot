# Sovellusohjelmointi 3 — demot

Tämä repositorio sisältää Sovellusohjelmointi 3 -opintojakson demoprojektit.

---

## Demo 3 — Tiedostojen upload

Kolmannessa demossa toteutetaan tiedostojen lähetys palvelimelle. Demo kattaa tiedostojen vastaanottamisen Express-palvelimella `multer`-middlewarella, tiedoston tallentamisen palvelimen tiedostojärjestelmään sekä tiedoston sijainnin palauttamisen asiakkaalle JSON-vastauksena.

## Demo 4 — React Native -perusteet

Neljännessä demossa tutustutaan React Native -sovelluskehykseen ja Expo-työkaluihin. Demo kattaa projektin luomisen `create-expo-app`-komennolla, React Nativen peruskomponentit (`View`, `Text`, `Image`, `TextInput`, `Pressable`, `FlatList`, `ScrollView`) ja niiden erot web-Reactiin sekä tyylien määrittämisen `StyleSheet`-objektilla. Sovellusta testataan fyysisellä laitteella Expo Go -sovelluksen kautta.

## Demo 5 — Laitekomponentit

Viidennessä demossa tutustutaan Expon laitekomponentteihin, joiden avulla päästään käsiksi mobiililaitteen ominaisuuksiin. Demosovellus näyttää laitteen perustiedot ja akun tilan sekä mahdollistaa puhelimen värisyttämisen.  Lisäksi otetaan käyttöön **React Native Paper** -komponenttikirjasto Material Design -käyttöliittymää varten.

## Demo 6 — Kamera

Kuudennessa demossa toteutetaan kuvaussovellus `expo-camera`-komponenttia käyttäen. Sovelluksessa käyttäjä voi ottaa useita kuvia laitteen kameralla, ja otetut kuvat näytetään aikaleimoineen vieritettävässä korttilistassa.
